### Building right now

**Origin** — a persistent VR multiverse for the Quest 3, running cyberpunk + autumn-shrine + future worlds inside a single streamed scene. UE 5.7, World Partition, HLOD, PSO precaching, and a custom loader stack tuned for VR comfort.

**Chakra** — the upcoming hardware demo for Origin: full-body haptic suit + foot-tracking shoe + Quest 3, shipping early July 2026.

**Oasis AI** — 25 autonomous LLM agents living in a Cyberpunk hub city. Each agent has needs, memory, vision, and a goal loop, all driven by a self-hosted Qwen2.5-72B and tied into Unreal via a bridge I wrote.
